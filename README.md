# start

your coding agent

## Setup

```bash
pnpm install
```

Sign in to Pi or configure your API keys before running the app.

## Run

```bash
pnpm dev
```

## Check

```bash
pnpm check
```

## Build

```bash
pnpm build
```

## Package

```bash
pnpm package
```

Create distributable builds:

```bash
pnpm dist
```
